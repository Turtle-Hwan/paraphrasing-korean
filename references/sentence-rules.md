# 문장 간소화 룰북

> **영감을 준 책:** [『내 문장이 그렇게 이상한가요?』](https://product.kyobobook.co.kr/detail/S000001863138)
> (저자 김정선, 유유, 2016).
>
> 이 룰북은 위 책에서 **영감을 받아** 만들었다. 어떤 군더더기를 다룰지는 책을 참고했지만,
> 아래 예문은 모두 규칙을 보여 주려고 **직접 지은 것**이다. 책의 문장이나 예문을 옮겨 싣지 않았다.
> 규칙은 한국어 어법 일반에 속하는 내용이다.

## 이 룰북의 정신

첨삭의 기본은 **빼기**다. 한 글자를 더해서 문장이 또렷해지면 남기고, 어수선해지면 덜어 낸다.
문장의 주인은 글쓴이가 아니라 **주어와 술어**다. 군더더기 수식·조사·접사·의존명사를 덜어
주어와 술어가 또렷이 드러나게 한다. 뜻은 한 글자도 바꾸지 않는다. 자세히는 [prime-directives.md].

심각도: **S1** = 거의 항상 빼거나 고친다 · **S2** = 문맥 보고 다듬는다.

---

## S-1. 적·의·것·들 [S1]

군더더기 네 가지 — 접미사 `-적`, 관형격 조사 `-의`, 의존명사 `것`, 복수 접미사 `-들`.
없어도 뜻이 통하면 뺀다. 이 책이 널리 알린 규칙이다.

- **-적**: 군더더기 `-적`은 명사를 그대로 쓴다.
  - 환경**적** 요인 → 환경 요인
  - 기술**적** 문제 → 기술 문제
- **-의**: 겹치거나 어색한 `-의`는 푼다. `-의` 자체가 번역 문체는 아니니 어색한 것만 손본다.
  - 회사**의** 성장 → 회사 성장
  - 교사와**의** 상담이 필요하다 → 교사와 상담할 필요가 있다
- **것**: `것이다/것은/것을`이 습관처럼 붙으면 명사·동사로 환원한다.
  - 운동하는 **것이** 중요하다 → 운동이 중요하다
  - 포기하지 않는 **것이** 답이다 → 포기하지 않아야 한다
- **-들**: 수가 이미 분명하면 복수 표시를 뺀다.
  - 여러 학생**들이** 모였다 → 여러 학생이 모였다
  - 책상 위에 책**들이** 쌓였다 → 책상 위에 책이 쌓였다

## S-2. 군더더기 "있다" / "-어 있는" [S1]

상태를 나타내는 `-어 있는/있다`, 틀 짓는 `-에 있어`·`-있을 예정` 등은 관형형으로 줄이거나 뺀다.

- 벽에 걸려 **있는** 그림 → 벽에 걸린 그림
- 길이 막혀 **있는** 상태다 → 길이 막힌 상태다
- 협상에 **있어** 신뢰가 중요하다 → 협상에는 신뢰가 중요하다
- 발표가 **있을** 예정이다 → 발표할 예정이다

## S-3. 이중피동·군더더기 피동 [S1]

이미 피동인 말에 `-이-`나 `-어지다`를 덧대지 않는다. 올바른 피동 한 번만 쓴다.

- 닫**혀진** 문 → 닫힌 문
- 쓰**여진** 글 → 쓰인 글
- 놓**여진** 책 → 놓인 책
- 끊**겨진** 전화 → 끊긴 전화
- 보**여지는** 풍경 → 보이는 풍경
- 활용 오류: 설레이는 → 설레는 · 개여서 → 개어

## S-4. "-에 의한/의해" → 능동 또는 "-에 따른/때문에" [S1]

영어 `by`-행위자 번역 문체다. 행위자를 주어로 돌리거나 `-에 따른`, `-로`, `때문에`로 바꾼다.

- 감독**에 의해** 발탁된 선수 → 감독이 발탁한 선수
- 사고**에 의한** 정체 → 사고로 생긴 정체
- 투표**에 의해** 결정된다 → 투표로 결정한다

## S-5. "대한/대해" — 막연한 두루뭉술 표현 [S1]

영어 `about/regarding`의 흔적이다. 구체적인 동사나 목적격 조사로 바꾼다.

- 그 계획**에 대해** 설명했다 → 그 계획을 설명했다
- 환경**에 대한** 관심 → 환경을 향한 관심
- 실패**에 대한** 두려움 → 실패를 두려워하는 마음

## S-6. "-로부터" 분해 [S2]

영어 `from`의 직역이다. 뜻에 따라 `-에게 / -와 / -에서`로 나눈다. 이중 조사도 푼다.

- 선배**로부터** 조언을 들었다 → 선배**에게** 조언을 들었다
- 과거**로부터의** 교훈 → 과거**에서** 얻은 교훈
- 회사**로부터** 멀어졌다 → 회사**에서** 멀어졌다
- 불안**으로부터** 벗어나다 → 불안**에서** 벗어나다

## S-7. 조사 "에 / 에게", 방향 조사 [S2]

무생물에는 `-에`, 사람·동물에는 `-에게`를 쓴다. 방향·위치 조사를 정확히 맞춘다.

- 정부**에게** 건의했다 → 정부**에** 건의했다
- 회사**에게** 요청했다 → 회사**에** 요청했다
- 도서관**을** 다닌다 → 도서관**에** 다닌다
- 하늘**에** 연을 날렸다 → 하늘**로** 연을 날렸다

## S-8. 군더더기 관형사 "그 / 어떤 / 어느" [S2]

영어 관사 `the/a/any`의 흔적이다. 없어도 되면 뺀다.

- **그 어떤** 말도 통하지 않았다 → 어떤 말도 통하지 않았다
- **그** 자신의 잘못 → 자신의 잘못
- **그 어느** 곳에서도 → 어느 곳에서도

## S-9. 군더더기 "-ㄹ 수 있는/있다" [S1]

가능·능력이 요지가 아니면 양태 표현을 뺀다.

- 읽을 **수 있는** 책이 많다 → 읽을 책이 많다
- 기억할 **수 있게** 적어 둔다 → 기억하게 적어 둔다
- 느낄 **수 있었던** 변화 → 느낀 변화
- 성공할 **수 있는** 가능성 → 성공할 가능성

## S-10. "시작하다" 오용 [S1]

순간·감정 동사인 놀라다·긴장되다·어색하다에 `-기 시작했다`를 붙이지 않는다. 그냥 과거형으로 쓴다.

- 그 순간 긴장되**기 시작했다** → 그 순간 긴장됐다
- 분위기가 싸늘해지**기 시작했다** → 분위기가 싸늘해졌다
- 재고가 바닥나**기 시작했다** → 재고가 바닥났다

## S-11. "-시키다" 남용 [S2]

사동이 아닌데 한자어에 `-시키다`를 붙이지 않는다. `-하다`로 쓴다.

- 그를 설득**시켰다** → 그를 설득했다
- 환경을 개선**시켰다** → 환경을 개선했다

## S-12. 시제 중복 · "-던" 오용 [S2]

한 절에 과거 표지를 거듭 쓰지 않는다. 완료된 동작은 `-ㄴ` 관형형으로 쓴다.

- 예전에는 더 노력**했었어야 했다** → 예전에는 더 노력해야 했다
- 지난주에 읽**었던** 책 → 지난주에 읽은 책

## S-13. "-는가/-인가" → "-는지/-인지" — 안긴 절 [S2]

내포된 절에서는 의문형 `-는가/-인가`가 아니라 `-는지/-ㄹ지`를 쓴다.

- 왜 실패했**는가를** 분석했다 → 왜 실패했**는지** 분석했다
- 그것이 옳은**가를** 따졌다 → 그것이 옳은**지** 따졌다

## S-14. 만연체 추리기 [S2]

`-고`·`-며`·`-는데`·`-지만` 같은 연결어미로 한없이 잇는 긴 문장은, 곁가지 절과 군더더기 수식을 덜어
한 문장 안에서 짧게 추린다(새 문장으로 쪼개지 않는다). 핵심 절만 남긴다.

- 회의가 길어졌고, 사람들이 지쳤으며, 결론도 안 났고, 시간만 흘렀다 → 회의가 길어져 다들 지쳤고 결론은 못 냈다

## S-15. 괄호 줄이기 [S1]

괄호로 설명을 덧붙이는 버릇을 줄인다. 괄호 안 내용을 문장으로 풀거나, 본문에 자연스럽게 녹이거나,
정말 필요 없으면 뺀다. 괄호는 꼭 필요할 때만 쓴다. 짧은 보충이면 풀어 쓴 한 문장이 낫다.

- 이 방법은 빠르고(속도가 강점이다) 비용도 낮다 → 이 방법은 빠르고 비용도 낮다
- 사용자(특히 초보자)는 헷갈린다 → 초보 사용자는 헷갈린다
- 그는 결국(여러 시도 끝에) 성공했다 → 그는 여러 번 시도한 끝에 성공했다
- 예외도 있다(드물지만) → 드물지만 예외도 있다

## S-16. 쉼표 줄이기 [S1]

쉼표를 최대한 아낀다. 쉼표로 길게 잇기보다 연결어미로 한 문장 안에서 푼다(쪼개지 않는다). 한 문장에 쉼표가 둘 넘으면
대개 너무 길다. 나열·삽입·도치로 쉼표가 쌓이면 그 문장을 다시 짠다.

- 그는 빠르게, 그리고 정확하게, 일을 끝냈다 → 그는 빠르고 정확하게 일을 끝냈다
- 우리는, 비록 시간이 없었지만, 끝까지 해냈다 → 우리는 시간이 없었지만 끝까지 해냈다
- 자료를 모으고, 정리하고, 분석해서, 결론을 냈다 → 자료를 모아 정리하고 분석해 결론을 냈다

## S-17. 에두르는 군더더기 "~축에 들다 / ~하는 편이다" [S2]

`~축에 들다`·`~라고 할 수 있다`처럼 단언을 에둘러 늘이는 표현이다. 군더더기면 빼고 곧장 단언한다.

- 가장 강한 이력서 축에 들어요 → 가장 뛰어난 이력서예요
- 잘하는 축에 든다 → 잘하는 편이다
- 효과가 있다고 할 수 있다 → 효과가 있다

## S-18. 풀어 쓴 표현을 적확한 말로 [S2]

뜻은 같은데 길게 에둘러 푼 표현을, 적확한 한 마디로 줄인다. **한 문장 안에서** 어휘만 바꾸며,
문장을 합치거나 내용을 빼지는 않는다(그건 교정·교열이 아니라 재집필이다).

- 영어를 옮긴 듯한 말투 → 영어 번역체
- 매우 빠른 속도로 처리한다 → 빠르게 처리한다
- 글을 읽는 사람의 입장에서 → 독자 입장에서

## 적용 순서 — 문장 간소화 단독 실행

문장마다: S-1 적·의·것·들 → S-2 있다 → S-3 이중피동 → S-9 수 있는 → S-10 시작하다
→ 나머지 S-4·5·6·7·8·11·12·13·17·18 → S-15 괄호·S-16 쉼표 → S-14 만연체는 마지막에 한 문장 안에서 군더더기를 줄인다.
뜻·수치·고유명사·인용 보존은 [prime-directives.md] 를 따른다.
