# 번역 문체 룰북 — 한국어로 자연스럽게

> 주로 영어를 직역한 듯한 흔적을 걷어 낸다. 문장 간소화 룰북 [sentence-rules.md] 과 겹치는 피동·의해·
> 대해·로부터는 거기에도 있다. 단독 명령(`:trans`·`:sentence`)이 각자 자기완결로 잡도록 **양쪽에 두되**,
> 여기서는 번역 문체에서 자주 나오는 표현을 큰 범주로 묶어 다룬다. 무생물 주어, 지나친 피동, 대명사 직역,
> 군더더기 `-들`, 관계절 직역, 가지다·만들다식 직역, 조사 겹침, 영문법 구문 직역을 정리했다.

심각도: **S1** = 거의 항상 고친다 · **S2** = 문맥 보고. 한 범주가 여러 버릇을 묶으므로 안에서도 강도가 다를 수 있다.

---

## T-1. 영어식 문장 골격 — 피동·사역·무생물 주어 [S1]

영어 수동태·사역·무생물 주어 구문을 한국어답게 푼다. 행위자를 주어로 세우고, 원인은 부사절로, 사역은 자동사·형용사로 돌린다.

- (피동→능동) 소녀가 뱀파이어**에게 피를 빨아 먹혔다** → 뱀파이어가 소녀의 피를 빨아먹었다
- (피동→능동) 노견은 시민협회**에 의해 보호되었다** → 시민협회가 그 노견을 보호했다
- (피동→능동) 회의가 위원회**에 의해 개최되었다** → 위원회가 회의를 열었다
- (사역 만들다) 세상을 더 따뜻하게 **만들었습니다** → 세상이 더 따뜻해졌습니다
- (사역 만들다) 그 소식이 나를 행복하게 **만들었다** → 그 소식에 나는 행복했다
- (무생물 주어) **이 그래프는** 매출 증가를 **보여 준다** → 이 그래프를 보면 매출이 늘었다
- (무생물 주어) **연구는** ~라고 **말한다** → 연구에 따르면 ~다
- (무생물 주어) **상황이** 나를 그렇게 **만들었다** → 그런 상황이라 그렇게 됐다
- (무생물 주어) **데이터가** 급증을 **나타낸다** → 데이터를 보면 급격히 늘었다

## T-2. "가지다 · -이 있다" 직역 (have / there is) [S1]

영어 have·possess·there is를 직역한 표현이다. 소유·특성은 형용사나 `있다`로, 동작성 명사는 동사로, 존재문은 술어로 환원한다.

- (가지다) 강한 경쟁력을 **가지고 있다** → 경쟁력이 강하다
- (가지다) 그녀는 두 명의 딸을 **가지고 있다** → 그녀에게는 딸이 둘 있다
- (가지다) 와인색 벽돌을 **가진** 집 → 와인색 벽돌집
- (가지다) 회의를 **가지다** → 회의를 열다
- (가지다) 위원회가 결정을 **내렸다** → 위원회가 결정했다
- (there is) 많은 문제들이 **있다** → 문제가 많다
- (there is) 거기에는 다양한 의견이 **있었다** → 의견이 다양했다

## T-3. 영어 전치사 직역 [S1]

about·through·for·from·with·in·case of 등을 전부 정형 조사로 옮기지 않고, 구체 동사나 알맞은 조사로 분산한다.

- (-로 인하여→때문에) 헛소문**으로 인해** 곤란에 빠졌다 → 헛소문 **때문에** 곤란해졌다
- (-로 인하여) 폭우**로 인하여** 행사가 취소되었다 → 폭우로 행사가 취소됐다
- (-에 대하여→은/를) 저의 장점**에 대해** 말씀드리자면 → 저의 장점은
- (-에 관하여) 그 사건**에 관하여** 조사 중이다 → 그 사건을 조사하고 있다
- (-을 통해→-로/-해서) 데이터 분석**을 통해** 인사이트를 얻는다 → 데이터를 분석해 인사이트를 얻는다
- (-을 통해) 대화**를 통해** 해결했다 → 대화로 해결했다
- (-을 위하여→-려고) 예약을 하기 **위해** 전화를 걸었다 → 예약을 하려고 전화를 걸었다
- (-을 위하여) 건강을 **위하여** 운동한다 → 건강하려고 운동한다
- (-로부터→-에서/-에게) 상사**로부터** 인정받았다 → 상사**에게** 인정받았다
- (-로부터) 한강**으로부터** 10킬로미터 → 한강**에서** 10킬로미터
- (-와 함께→-며) 미소**와 함께** 인사했다 → 웃으며 인사했다
- (-와 함께) 커피**와 함께** 책을 읽었다 → 커피를 마시며 책을 읽었다
- (-에 있어→-에서/-할 때) 인생**에 있어** 가장 중요한 것 → 인생에서 가장 중요한 것
- (-에 있어) 성능을 높이는 데**에 있어** → 성능을 높일 때
- (-의 경우→-은/-에는) 저**의 경우**에는 매일 운동합니다 → 저는 매일 운동합니다
- (-의 경우) 주말**의 경우** 사람이 많다 → 주말에는 사람이 많다

## T-4. 기능어 군더더기 — 대명사·복수 -들·잉여 접속·부정관사 [S1]

영어 대명사·복수 -s·and·부정관사를 직역한 군더더기다. 한국어는 대명사를 자주 생략하고, 수 표지·잉여 접속·관사를 덜어 낸다. 문장 첫머리의 접속사는 AI 문체 AI-7로 본다.

- (대명사) 존은 피곤했다. **그는** 자리에 앉아 한숨을 쉬었다 → 존은 피곤했다. 자리에 앉아 한숨을 쉬었다
- (대명사) 메리는 **그녀의** 어머니에게 전화했다 → 메리는 어머니에게 전화했다
- (대명사) they → 그들 대신 사람들·우리·일부로 다양화
- (복수 -들) 많은 학생**들이** 있다 → 많은 학생이 있다
- (복수 -들) 나무**들**과 꽃**들이** 가득한 정원 → 나무와 꽃이 가득한 정원
- (잉여 접속) 그는 책을 읽었고 **그리고** 잠이 들었다 → 그는 책을 읽다가 잠이 들었다
- (잉여 접속) 빠르고 **그리고** 정확하며 **또한** 저렴하다 → 빠르고 정확한 데다 값까지 싸다
- (부정관사) 그것은 **하나의** 좋은 예다 → 그것은 좋은 예다
- (부정관사) 이것은 **하나의** 도전이다 → 이것은 도전이다

## T-5. "-의" 남용·연쇄 · 명사화 [S1]

영어 of·일본어 の를 옮긴 군더더기 `-의`와, 동사로 풀 것을 `~함/~음/~기`로 굳힌 명사화를 푼다.

- (-의 연쇄) 최고의 인기를 누리는 가수 → 가장 인기 있는 가수
- (-의 연쇄) 그녀의 두 명의 딸의 사진 → 그녀의 두 딸 사진
- (명사화) 환경 보호에 대한 인식의 제고가 필요하다 → 환경을 더 잘 지켜야 한다는 인식을 키워야 한다
- (명사화) 문제의 해결을 위한 방안의 모색 → 문제를 풀 방안을 찾기

## T-6. 영문법 구문 직역 — 가주어·비교·정도·조건·관계절 [S2]

가주어, one of, 비교급, as~as, so~that, when·if, 이중부정, 관계절 좌향 수식처럼 영문법 틀을 그대로 옮긴 구문이다. 한국어 어순·어미로 푼다.

- (가주어) 그가 거짓말했다는 **것은** 분명하다 → 그는 분명히 거짓말했다
- (가주어) 우리가 손을 씻는 **것이** 필요하다 → 손을 씻어야 한다
- (~중 하나) 그는 가장 뛰어난 선수들 **중 하나**다 → 그는 손꼽히는 선수다
- (~중 하나) 그곳은 가장 아름다운 도시 **중 하나**다 → 그곳은 손꼽히게 아름다운 도시다
- (비교급 보다 더) 그는 나보다 **더** 키가 크다 → 그는 나보다 키가 크다
- (비교급) 무엇보다도 **더** 중요한 것 → 무엇보다 중요한 것
- (~만큼) 그녀는 천사**만큼** 친절하다 → 그녀는 천사처럼 친절하다
- (~만큼) **가능한 한** 빨리 → 되도록 빨리
- (너무~해서) 너무 피곤**해서** 일어날 **수 없었다** → 너무 피곤해서 못 일어났다
- (너무~해서) 너무 빨리 말**해서** 알아들을 **수 없었다** → 너무 빨리 말해 못 알아들었다
- (when/if 부사절) 집에 도착했**을 때** 비가 오고 있었다 → 집에 도착하니 비가 왔다
- (when/if 부사절) **만약** 당신이 원한**다면** 도와줄게요 → 원하시면 도와드릴게요
- (이중부정) 감탄하지 **않을 수 없었다** → 절로 감탄했다
- (이중부정) 노력 없이는 성공할 **수 없다** → 노력해야 성공한다
- (관계절 좌향 수식) 어제 내가 도서관에서 빌린 그 책을 읽고 있던 친구가 말을 걸었다 → 어제 도서관에서 빌린 책을 읽던 친구가 말을 걸었다

## T-7. 시제 직역 — 진행형·완료 [S2]

영어 진행형·완료를 기계적으로 늘여 쓴 표현이다. 단순 현재·과거로 충분하면 줄인다.

- (진행형) 전철이 도착하는 **중입니다** → 전철이 도착합니다
- (진행형) 점점 나아지고 있는 **중이다** → 점점 나아진다
- (완료) 그동안 많은 노력을 기울여 **왔다** → 그동안 많이 노력했다
- (완료) 나는 항상 너를 사랑해 **왔다** → 나는 늘 너를 사랑한다

## T-8. 일본어투 조사·서술 틀 · 행정어 [S2]

일본어투에서 온 굳은 조사·서술 틀과, 공문서의 어려운 한자어·행정어를 쉬운 말로 푼다. 법령·계약처럼 정해진 문구는 그대로 둔다. 쉬운 공공언어는 [style-guide.md] ST-7도 함께 본다.

- (일본어투) 규정에 의거하여 처리한다 → 규정에 따라 처리한다
- (일본어투) 참가자에 한하여 입장할 수 있다 → 참가자만 입장할 수 있다
- (일본어투) 검토를 요한다 → 검토해야 한다
- (일본어투) 회의를 갖도록 한다 → 회의를 연다
- (행정어) 금번 회의 → 이번 회의
- (행정어) 상기 내용을 참고 바랍니다 → 위 내용을 참고해 주세요
- (행정어) 당해 사업의 추진 → 해당 사업 추진
- (행정어) 추후 별도 공지 → 나중에 따로 알림

## T-9. 자잘한 번역 문체 치환 [S2]

- ~에 위치하다 → ~에 있다
- ~하게 만들었다 → ~해졌다 · ~하게 했다
- ~에도 불구하고 → ~지만 · ~는데도
- ~하는 중이다 → ~하고 있다 · ~한다
- ~에 다름 아니다 → 바로 ~다
- ~을 필요로 한다 → ~이 필요하다

## 적용 순서 — 번역 문체 단독 실행

문장마다: T-1 문장 골격(피동·사역·무생물 주어) → T-2 가지다·있다 → T-3 전치사 직역
→ T-4 기능어 군더더기 → T-5 -의·명사화 → T-6 영문법 구문 → T-7 시제 → T-8 일본어투·행정어
→ T-9 자잘한 치환. 뜻·수치·고유명사·인용 보존은 [prime-directives.md] 를 따른다.
쉬운 공공언어는 [style-guide.md] 도 함께 본다. 근거 지도는 [sources.md].
