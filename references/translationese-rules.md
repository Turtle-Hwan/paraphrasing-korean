# 번역투 룰북 — 한국어로 자연스럽게

> 주로 영어를 직역한 듯한 흔적을 걷어 낸다. 센텐스 룰북([sentence-rules.md])과 겹치는 항목
> (피동·의해·대해·로부터)은 거기에 두고, 여기서는 **번역투에서만 나오는 표현**을 다룬다.
> 무생물 주어, 지나친 피동, 대명사 직역, 군더더기 `-들`, 관계절 직역, "가지다·만들다"식 직역,
> 조사 겹침 같은, 한국어 번역에서 자주 지적되는 버릇들을 실용 규칙으로 정리했다.

심각도: **S1** = 거의 항상 고친다 · **S2** = 문맥 보고.

---

## T-1. 피동태 → 능동 [S1]

영어 수동태를 그대로 옮긴 피동. 행위자를 주어로 세워 능동으로.

- 소녀가 뱀파이어**에게 피를 빨아 먹혔다** → 뱀파이어가 소녀의 피를 빨아먹었다
- 노견은 시민협회**에 의해 보호되었다** → 시민협회가 그 노견을 보호했다
- 회의가 위원회**에 의해 개최되었다** → 위원회가 회의를 열었다

## T-2. "가지다" (have) 직역 [S1]

영어 `have/possess`. 소유·특성은 형용사나 `있다`로, 동작성 명사는 동사로 환원.

- 강한 경쟁력을 **가지고 있다** → 경쟁력이 강하다
- 그녀는 두 명의 딸을 **가지고 있다** → 그녀에게는 딸이 둘 있다
- 와인색 벽돌을 **가진** 집 → 와인색 벽돌집
- 회의를 **가지다** → 회의를 하다(열다)
- 위원회가 결정을 **내렸다(made a decision)** → 위원회가 결정했다

## T-3. "-로 인하여 / -로 인해" 간결화 [S1]

원인은 `때문에`, `-로`로 줄인다.

- 헛소문**으로 인해** 곤란에 빠졌다 → 헛소문 **때문에** 곤란해졌다
- 폭우**로 인하여** 행사가 취소되었다 → 폭우로 행사가 취소됐다

## T-4. "-에 대하여 / -에 관하여" (about/regarding) [S1]

주제는 `-은/는`, `-을/를`, 또는 구체 동사로.

- 저의 장점**에 대해** 말씀드리자면 → 저의 장점은
- 환경 문제**에 대해** 발표하겠습니다 → 환경 문제를 발표하겠습니다
- 그 사건**에 관하여** 조사 중이다 → 그 사건을 조사하고 있다

## T-5. 무생물·추상 주어 → 부사구 / 사람 주어 [S1]

영어식 무생물 주어 + 타동사를 한국어답게 푼다. 원인은 부사절로, 주체는 사람으로.

- **이 그래프는** 매출 증가를 **보여 준다** → 이 그래프를 보면 매출이 늘었다
- **연구는** ~라고 **말한다(주장한다)** → 연구에 따르면 ~다
- **상황이** 나를 그렇게 **만들었다** → 그런 상황이라 그렇게 됐다
- **데이터가** 급증을 **나타낸다(show)** → 데이터를 보면 급격히 늘었다

## T-6. "-을 통해 / 통하여" 남발 (through/via) [S1]

수단·경로를 전부 `통해`로 처리하지 않는다. `-로`, `-해서`, `-함으로써`로 분산.

- 데이터 분석**을 통해** 인사이트를 얻는다 → 데이터를 분석해 인사이트를 얻는다
- 대화**를 통해** 해결했다 → 대화로(대화해서) 해결했다

## T-7. 영어 대명사 직역 (그/그녀/그것/그들) [S1]

한국어는 대명사를 자주 생략한다. 절반 이상은 빼고, 필요한 곳만 명사·호칭으로.

- 존은 피곤했다. **그는** 앉았다. **그는** 한숨을 쉬었다 → 존은 피곤했다. 자리에 앉아 한숨을 쉬었다
- 메리는 **그녀의** 어머니에게 전화했다 → 메리는 어머니에게 전화했다
- `they` → 그들 대신 사람들·우리·일부로 다양화

## T-8. 관계절 좌향 수식 과다 [S2]

영어 후치 수식을 그대로 길게 앞에 붙이면 숨차다. 두 문장으로 풀거나 순서를 바꾼다.

- 어제 내가 도서관에서 빌린 그 책을 읽고 있던 친구가 → 친구가 그 책을 읽고 있었다. 어제 내가 도서관에서 빌린 책이다.

## T-9. 자잘한 번역투 치환 [S2]

- ~에 위치하다 → ~에 있다
- ~하게 만들었다 → ~해졌다 / ~하게 했다
- ~에도 불구하고 → ~지만 / ~는데도
- ~하는 중이다 → ~하고 있다 / ~한다
- ~에 다름 아니다 → 바로 ~다
- ~을 필요로 한다 → ~이 필요하다

---

## 적용 순서(번역투 단독 실행 시)

문장마다: T-1(피동) → T-2(가지다) → T-5(무생물 주어) → T-3·T-4·T-6(인하여·대해·통해)
→ T-7(대명사) → T-8(관계절) → T-9(자잘한 치환).
의미·수치·고유명사·인용 보존은 [prime-directives.md] 를 따른다.
