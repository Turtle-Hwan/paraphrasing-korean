# 맞춤법 룰북

> 한국어 맞춤법·문법 교정. 국립국어원 어문 규범과 표준국어대사전을 근거로 한다. 맞춤법 교정은
> 뜻을 바꾸지 않으니 모든 명령에서 항상 함께 본다.

심각도: **S1** = 명백한 오류라 거의 항상 고친다 · **S2** = 문맥·뜻에 따라 갈리니 보고 고친다.

---

## G-1. 서술격조사 '이다' 해요체 — 예요/이에요/에요 [S1]

받침이 없으면 `-예요`로 줄여 쓰고, 받침이 있으면 `-이에요`로 쓴다. `-에요`는 '아니다'의 '아니에요'에만 붙는다.

- 도구에요 → 도구**예요**
- 거에요 → 거**예요**
- 책이예요 → 책**이에요**
- 아니예요 → 아니**에요**

## G-2. 되 / 돼 [S2]

`돼`는 '되어'의 준말이다. '되어'로 풀어 말이 되면 `돼`, 안 되면 `되`로 쓴다. 자리에 '해'가 들어가면 `돼`,
'하'가 들어가면 `되`다.

- 잘 됬다 → 잘 **됐다**
- 그러면 안 되 → 그러면 안 **돼**
- 빨리 돼고 싶다 → 빨리 **되고** 싶다
- 내일 뵈요 → 내일 **봬요** — '뵈어요'의 준말이라 같은 원리다

## G-3. 안 / 않 [S2]

`안`은 부사라 띄워 쓰고, `않-`은 용언 어간이라 붙여 쓴다.

- 밥을 않 먹었다 → 밥을 **안** 먹었다
- 가지 안았다 → 가지 **않았다**
- 깨끗하지 안다 → 깨끗하지 **않다**

## G-4. -데 / -대 [S2]

`-데`는 내가 직접 겪은 일을 회상할 때, `-대`는 남의 말을 전할 때 쓴다.

- 거기 사람 많대 → 거기 사람 많**데**
- 걔 결혼한데 → 걔 결혼한**대**

## G-5. -든·-든지 / -던 [S2]

`-든·-든지`는 선택·무관, `-던`은 과거 회상이다.

- 가던지 말던지 → 가**든지** 말**든지**
- 어렸을 때 놀든 곳 → 어렸을 때 놀**던** 곳

## G-6. 로서 / 로써 [S2]

`-로서`는 자격·지위, `-로써`는 수단·도구다.

- 학생으로써 본분 → 학생**으로서** 본분
- 대화로서 해결 → 대화**로써** 해결

## G-7. -율 / -률, -열 / -렬 [S1]

받침이 없거나 'ㄴ' 받침 뒤면 `율`·`열`, 그 밖의 받침 뒤면 `률`·`렬`로 쓴다.

- 비률 → 비**율** · 확율 → 확**률**
- 나렬 → 나**열** · 정열 → 정**렬**

## G-8. 채 / 체 / 째 [S2]

`채`는 그대로의 상태, `체`는 '척', `째`는 차례·전부를 뜻한다.

- 신발을 신은 체 → 신발을 신은 **채**
- 못 본 채하다 → 못 본 **체**하다
- 껍질채 → 껍질**째** · 통**째**로

## G-9. 맞히다 / 맞추다 [S2]

`맞히다`는 정답·적중, `맞추다`는 비교·조정이다. 서로 대조하는 뜻이면 `맞추다`를 그대로 둔다.

- 정답을 맞추다 → 정답을 **맞히다**
- 과녁을 맞추다 → 과녁을 **맞히다**
- 답안지를 맞춰 보다 — 서로 대조하는 뜻이라 그대로 둔다

## G-10. 너머 / 넘어, 가르치다 / 가리키다 [S1·S2]

`너머`는 공간을 가리키는 명사, `넘어`는 동작이다. `가르치다`는 교육, `가리키다`는 지시다.

- 산 넘어 마을 → 산 **너머** 마을
- 고개를 너머 갔다 → 고개를 **넘어** 갔다
- 영어를 가리키다 → 영어를 **가르치다**
- 손가락으로 가르치다 → 손가락으로 **가리키다**

## G-11. 부치다 / 붙이다, 늘리다 / 늘이다 [S2]

'붙다' 뜻이 살면 `붙이다`, 멀면 `부치다`다. 수·양은 `늘리다`, 길이는 `늘이다`다.

- 우표를 부치다 → 우표를 **붙이다** · 편지를 붙이다 → 편지를 **부치다**
- 인원을 늘이다 → 인원을 **늘리다** · 고무줄을 늘리다 → 고무줄을 **늘이다**

## G-12. 낫다 / 낳다 / 났다 [S1]

`낫다`는 회복·우수, `낳다`는 출산·산출, `났다`는 '나다'의 과거다.

- 감기가 낳았다 → 감기가 **나았다**
- 아이를 나았다 → 아이를 **낳았다**
- 이게 더 낳다 → 이게 더 **낫다**

## G-13. 결제 / 결재, 지양 / 지향 [S2]

`결제`는 지불, `결재`는 승인이다. `지양`은 피함, `지향`은 추구다.

- 카드 결재 → 카드 **결제** · 서류 결제 → 서류 **결재**
- 폭력은 지향해야 한다 → 폭력은 **지양**해야 한다
- 평화를 지양하는 사회 → 평화를 **지향**하는 사회

## G-14. 굳어진 오타 모음 [S1]

자주 틀리지만 정답이 하나뿐인 표기.

- 됬→됐 · 몇일→며칠 · 역활→역할 · 설레임→설렘 · 오랫만→오랜만 · 오랜동안→오랫동안
- 희안→희한 · 구지→굳이 · 닥달→닦달 · 으례→으레 · 금새→금세 · 폭팔→폭발 · 어의없다→어이없다
- 바램→바람 · 잠궜다→잠갔다 · 담궜다→담갔다 · 왠만→웬만 · 안절부절→안절부절못하다

## G-15. -ㄹ게 / -ㄹ께, -려고 / -ㄹ려고 [S1]

약속 종결은 소리가 [께]라도 `-ㄹ게`로 쓴다. 의도는 `-려고`로 쓰고 `-ㄹ려고`는 비표준이다.

- 할께 → 할**게** · 갈께요 → 갈**게요**
- 갈려고 → **가려고** · 먹을려고 → **먹으려고**

## G-16. 왠지 / 웬 [S2]

'왜인지'를 줄인 `왠지`만 '왠'을 쓰고, 나머지는 모두 `웬`이다.

- 웬지 슬프다 → **왠지** 슬프다
- 왠일이야 → **웬일**이야 · 왠만하면 → **웬만하면**

## G-17. 어떻게 / 어떡해 [S1]

`어떻게`는 부사형이라 뒤에 서술어가 오고, `어떡해`는 '어떻게 해'의 준말이라 그 자체로 서술이 된다.

- 이걸 어떡해 하지 → 이걸 **어떻게** 하지
- 나 이제 어떻게 → 나 이제 **어떡해**

## G-18. -이 / -히 부사 [S2]

소리가 '이'로만 나면 `-이`, '히'로도 나면 `-히`로 쓴다. 첩어나 'ㅅ' 받침 뒤는 대개 `-이`다.

- 이로 적는 말: 깨끗히 → 깨끗**이** · 곰곰히 → 곰곰**이** · 일일히 → 일일**이**
- 히로 적는 말: 솔직이 → 솔직**히** · 꼼꼼이 → 꼼꼼**히**

## G-19. 띄어쓰기 — 조사 붙임, 의존명사 띄움 [S2]

조사는 앞말에 붙인다. 의존명사인 것·수·뿐·만큼·데·지는 띄운다.

- 할수 있다 → 할 **수** 있다 · 먹을것 → 먹을 **것**
- 아는데 → 아는 **데** · 만난지 3년 → 만난 **지** 3년
- 조사는 붙인다: 너 뿐 → 너**뿐** · 학교 에서 → 학교**에서**

## G-20. 띄어쓰기 — 단위·호칭, 안 되다 / 안되다 [S2]

수 뒤 단위명사는 띄우고, 성과 이름은 붙이되 호칭은 띄운다. 부정의 '안 되다'는 띄우고, 섭섭하거나
일이 잘 안 풀리는 '안되다'는 붙인다.

- 한개 → 한 **개** · 두살 → 두 **살** · 홍 길동 → **홍길동** · 홍길동씨 → 홍길동 **씨**
- 금지·부정은 띄운다: 거짓말하면 안된다 → 거짓말하면 **안 된다**
- 섭섭함은 붙인다: 마음이 안 됐다 → 마음이 **안됐다**
- 일이 잘 안 풀림은 붙인다: 사업이 안 된다 → 사업이 **안된다**
