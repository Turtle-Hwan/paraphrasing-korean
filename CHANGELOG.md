# 바뀜 기록

## Unreleased

분절 정확도를 높이고, 정규식 힌트를 '맞춤법 전용'으로 좁혔어요.

- **문장 다이어트 규칙 추가**: `sentence-rules.md`에 S-17(에두르는 군더더기 "~축에 들다")을
  더했어요. 실제 윤문에서 본 문장을 예시로 넣었어요.
- **윤문 문체 지향 추가**: `prime-directives.md` '한국어답게 쓰기'에 "어휘를 다양하게, 표현을
  우아하게"를 더했어요. 적확한 낱말로 수려하게 — 과윤문은 경계.
- **분절 보정**: 마침표 없는 줄바꿈 글을 줄 단위로 쪼개고(무종결), 약어(A.I.·e.g.)·숫자(1980.)의
  마침표를 문장 끝으로 오인하지 않으며, 소프트랩된 문장의 워크시트 원문을 한 줄로 접어요. 무손실 유지.
- **정규식 힌트 축소**: `scan.py` 힌트를 고정밀 맞춤법·표기 오타(예요/에요·됬·역활·-ㄹ께)만 남기고,
  문체·번역투·AI 티 패턴은 정규식에서 뺐어요. 정규식이 느슨해 오탐이 많고 신호를 희석시켰거든요.
  이런 패턴은 규칙 글(룰북)의 before→after 예시로 에이전트가 직접 판단해요(in-context learning).

## 0.3.0 — 2026-06-16

맞춤법과 문체 교정을 더하고, 한국어 특화를 분명히 했어요.

- **맞춤법 교정 추가**: `grammar-rules.md`(G-1~G-20). 예요/이에요, 되/돼, 안/않, 데/대, 든/던, 율/률,
  띄어쓰기 등 국립국어원 어문 규범 기준.
- **문체 교정 추가**: `style-guide.md`(ST-1~ST-8). 한 글에서 종결문체(해요체/합니다체/한다체) 섞지 않기,
  사물존칭 제거, 간접높임 보존, 객체높임 어휘.
- **새 명령 `-grammar`**: 맞춤법·문체만 따로 교정. 통합 명령은 맞춤법 → 번역 문체 → AI 문체 → 문장 다이어트
  → 문체 순서로 적용.
- **AI 문체 보강**: AI-27~AI-31. 어휘 다양성 부족, AI 군더더기 어휘, 과격식 톤, 담화표지 부재, 환각 출처.
  박종향·김은영 「생성형 AI 텍스트와 인간 텍스트의 비교 연구」(교양교육연구, 2025) 등 논문 근거.
- **README 개편**: 한국어 특화·국립국어원/논문 근거·LLM이 한 줄씩 읽는 방식을 앞세웠어요.

## 0.2.0 — 2026-06-16

설치를 쉽게 하고 규칙을 더했습니다.

- **한 줄 설치**: Claude Code 마켓플레이스 설정을 넣었습니다. `/plugin marketplace add` 한 번이면 됩니다.
- **설치 스크립트 강화**: 설치돼 있는 도구를 스스로 찾고, 충돌 시 백업하고, `--dry-run`·`--copy` 를 지원합니다.
  `update.sh` 와 `uninstall.sh` 도 넣었습니다.
- **Gemini CLI 지원**: `gemini-extension.json` 과 `GEMINI.md`, 그리고 `commands/` 를 넣었습니다.
- **새 규칙**: 번역 문체 T-10~T-30, AI 문체 AI-10~AI-26 을 더했습니다. 괄호 줄이기 S-15 와 쉼표 줄이기 S-16 도
  넣고, 공통 약속에 "한국어답게 쓰기"를 더했습니다.
- **이름 정리**: 문장 다이어트 명령을 `-sentence` 로 통일했습니다.
- **문서 정리**: 한국어 특화를 앞세우고, 괄호와 쉼표를 줄여 더 자연스럽게 다시 썼습니다. 책 인용을
  『내 문장이 그렇게 이상한가요?』(저자 김정선, 유유, 2016)로 통일했습니다.

## 0.1.0 — 2026-06-15

첫 공개. 한국어 글을 문장 하나씩 다듬어 주는 스킬 묶음.

- **명령어 4개**: `paraphrasing-korean` 통합, `-sentence` 문장 다이어트, `-trans` 번역 문체, `-ai` AI 문체.
- **문장 단위로만 다듬기**: `segment.py` 가 문장 부호로 잘라 작업표를 만들고, `reassemble.py` 가 되붙이며
  문장 수와 변경량을 검사합니다. 정규식으로 한꺼번에 바꾸지 않고 문장 칸을 하나씩 읽고 고치도록 강제합니다.
- **정규식 힌트**: `scan.py` 가 의심 문장에 규칙 번호를 달아 줍니다. 참고용이고 고치는 일은 에이전트가 합니다.
- **규칙 글 4종**: `sentence-rules`, `translationese-rules`, `ai-tell-rules`, `prime-directives`.
- **여러 도구 지원**: Claude Code·Codex·OpenClaw·Hermes. Claude Code 플러그인 설정도 포함합니다.
